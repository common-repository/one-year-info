1年以上前の記事に対して注意を促します。

== Description ==

1年以上前の記事にアクセスされた場合、「更新日から1年以上経過しています。情報が古い可能性がございます。」と表示されます。

== Installation ==

1. Upload the entire `one-year-info` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
